# Лабораторная работа №1 - Перемножение матриц с использованием CUDA

## Описание

В данной работе были реализованы два метода умножения матриц: последовательное вычисление значений на CPU и параллельное на GPU с использованием CUDA - ядер, а также проведено сравнение времени выполнения и оценка эффективности распараллеливания вычислений.

## Принцип умножения матриц на CPU

В функции compute_matmul_CPU() выполняется последовательное вычисление элементов результирующей матрицы. С помощью тройного вложенного цикла происходит перемножение элементов матриц A и B и запись результатов в матрицу C. Данная реализация имеет кубическую временную сложность (O(N^3)), что сильно влияет на скорость вычислений при больших размерностях матриц.

## Принцип умножения матриц на GPU

1. Копирование значений матриц A и B из CPU в GPU.
2. Конфигурация количества потоков в блоке (16x16 потоков) и количества блоков в сетке
2. Запуск CUDA - ядра и функции compiled_kernel для выполнения умножения матриц.
3. Копирование результата умножения из GPU в CPU.
4. Освобождение памяти в GPU.

Каждый поток обрабатывает одну строку и один столбец первой и второй исходных матриц соответственно. таким образом, происходит распараллеливание операций: каждый поток вычисляет конкретный элемент результирующей матрицы.

## Выводы

Благодаря распараллеливанию вычислений на большом числе потоков удалось добиться существенного повышения скорости перемножения матриц на GPU. Увеличение преимущества видеоускорителя по сравнению с процессором для математических операций пропорционально росту размерности матриц и достигает максимума при размерности 1000x1000.
